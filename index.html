Informatikkenntnisse

HTML

HTML ist das Grundgerüst jeder Webseite. Es wird verwendet, um den Inhalt einer Seite zu strukturieren, wie Texte, Bilder und Links. Es ist sozusagen das Skelett einer Webseite, das bestimmt, was wo steht.

Learn more

JS

JavaScript ist eine Programmiersprache für dynamische und interaktive Webseiten. Es ermöglicht Funktionen wie aufklappbare Menüs, Formularvalidierung und das Ändern von Inhalten ohne Neuladen der Seite.

Learn more

CSS

CSS ist verantwortlich für das Aussehen und die Gestaltung einer Webseite. Mit CSS kann man das Design anpassen, Farben, Schriftarten und Layouts festlegen.

Learn more

SQL

SQL (Structured Query Language) ist eine Programmiersprache, die verwendet wird, um Daten in relationalen Datenbanken zu verwalten. Mit SQL kann man Daten abfragen, einfügen, aktualisieren und löschen sowie Datenbankschemata erstellen und ändern.

Learn more

Python

Python ist eine einfache, vielseitige Programmiersprache, die in vielen Bereichen wie Webentwicklung, Datenanalyse und KI eingesetzt wird. Sie hat eine klare Syntax und viele nützliche Bibliotheken.
